# Device smoke-test

Заполнять только на физическом iPhone после сборки и подписи IPA.

## Среда

- [ ] модель iPhone записана;
- [ ] версия iOS записана;
- [ ] способ подписи и установки записан;
- [ ] commit приложения записан;
- [ ] SHA-256 IPA записан;
- [ ] версия NodeMobile записана.

## Runtime

- [ ] приложение запускается без crash;
- [ ] в логах есть `ST_CONTROL_READY`;
- [ ] в логах есть `ST_SERVER_READY`;
- [ ] показанная версия Node совпадает с артефактом;
- [ ] сервер слушает только `127.0.0.1`;
- [ ] порт 8000 работает;
- [ ] при занятом 8000 выбирается 8001;
- [ ] stop закрывает content port;
- [ ] start открывает port заново;
- [ ] restart меняет port;
- [ ] stdout и stderr отображаются.

## Клиент

- [ ] встроенный WKWebView открывает smoke page;
- [ ] health endpoint отвечает;
- [ ] после блокировки/разблокировки состояние корректно обновляется;
- [ ] отдельно записано поведение при открытии Safari;
- [ ] отдельно записано время до suspension в Safari;
- [ ] после возврата из Safari health восстанавливается либо показывается ошибка.

## Данные

- [ ] UserData доступен после перезапуска приложения;
- [ ] лог сохраняется;
- [ ] ограничение размера лога работает;
- [ ] удаление/обновление приложения не заявляется как безопасное без отдельной проверки.

## Node 22 gate

- [ ] `typeof WebAssembly` записан;
- [ ] импорт `@jimp/wasm-png` проверен;
- [ ] импорт `tiktoken` проверен;
- [ ] импорт `src/jimp.js` проверен;
- [ ] импорт `src/server-startup.js` проверен;
- [ ] настоящий `server.js` SillyTavern запущен;
- [ ] главная страница SillyTavern открыта в WKWebView.
